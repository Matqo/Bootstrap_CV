No templates were used in creating this site

Bootstrap features used:

  • Glyphicons (See submit button for Hire me/Contact form)
  • Containers & Rows (See divs, fluid containers used initially however centered design was chosen later)
  • Dropdowns (See Navbar, when screen cannot display natural navbar, dropdown icon displayed)
  • Buttons (See Hire Me button, Send it form button or round buttons for social media icons)
  • Sizing (See col-md of varying sizes for best layout choices)
  • Forms (See Hire me/Contact form)
  • Static (See NavBar which is stuck to top of screen at all times)
  • Media tags (See css_matqo.css where Media tags are used to size fonts appropriately depending on screen size)
  • Table (See Education section in portfolio for bootstrap table of GPA per year)
  • XS divs (See Portfolio and Interests section where they're used as div separators, disappear on small screens)

JQuery features used:

  • JQuery Bootstrap Validation (Used on Bootstrap Hire me/Contact form to validate from any device)
  • JQuery BPopUp (Used on this window and Hire me/Contact form to hide un-necessary information from user)
  • JQuery easing (Used on BPopUp to soften animations)
  • JQuery Simple Text Rotator (Used on hero component to cycle in between "specialisations")
  • JQuery Isotope (Used on Porfolio image section, filtering and displaying with UI)
  • JQuery Navigation (Used on Navbar to provide easy navigation with style)
  • JQuery Smoothscroll (Used on Navbar scrolling, instead of just jumping to section)
  • JQuery Magnific Popup (Used on Portfolio section in order to display image library with description)
  • JQuery Scroll to (Used on Navbar section in order to scroll to specific div)
  • JQuery Wow (Used on majority of divs to reveal themselves as user scrolls through the site)

Fonts used:
